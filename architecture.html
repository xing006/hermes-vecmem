VecMem 架构图

Hermes Agent 向量记忆插件 — 基于 sqlite-vec

数据流

  • 对话 → sync_turn 提取事实 → 嵌入 → 存储
  • 下一轮 → prefetch 向量召回 → 注入 system prompt
  • memory 工具写入 → 自动镜像到向量库

三级降级

  • API 模式 (DeepSeek/OpenAI 嵌入)
  • 本地模式 (sentence-transformers)
  • Hash 降级 (永不崩溃)

混合检索

  • 向量搜索 — 语义匹配
  • FTS5 搜索 — 关键词精确匹配
  • 嵌入缓存 — 相同文本零成本命中
